## 1、zip





## 2、unzip
解压ZIP到当前文件夹，并返回文件名列表
```
unzip(zipfilename)
unzip(zipfilename,outputdir)
filenames = unzip(zipfilename,outputdir)
```
## 3、gzip
把文件压缩为.gz格式
```
gzip(files)
gzip(files,outputdir)
filenames = gzip(...)
```
## 4、 gunzip
解压.gz到当前文件夹(GNU)
```
gunzip(files)
gunzip(files,outputdir)
gunzip(url, ...)
filenames = gunzip(...)
```
## 5、tar
.tgz或.tar
```
tar(tarfilename,files)
tar(tarfilename,files,rootfolder)
entrynames = tar(...)

tar('backup.tgz','.');
```
