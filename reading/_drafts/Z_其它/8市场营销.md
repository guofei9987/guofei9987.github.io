# 行业势力度量指标
1. 勒纳指数 lerner Index
$LI=(P-MC)/P$
其中P是价格，MC是边际成本
LI介于0和1，为0时时完全竞争
2. 贝恩指数 Bain Index
$\Pi =R-C-D-iV$
R是收益，C是成本，D是可能的贬值额
iV是资本收益
3. Papandreou Index
需求交叉弹性
