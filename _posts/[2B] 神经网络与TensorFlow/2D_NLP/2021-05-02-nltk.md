## 词频统计

```python
from bs4 import BeautifulSoup
import requests

# 先用爬虫获取一段语料
text = BeautifulSoup(requests.get('https://www.python.org/').text, 'lxml').get_text()


import nltk

freq_dist = nltk.FreqDist(text.split())  # 类似 Counter 的词频统计
freq_dist.plot(50, cumulative=False)  # 画图
```
