## python 调用 C


1. 写 c 文件， 保存为 `file.c`
```c
#include<stdio.h>
void add()
{
int sum=0;
for(int i=0;i<1000;i++)
sum+=i;
printf("sum:%d\n",sum);
}
```
2. 把 `file.c` 文件编译为 `file.so`：`gcc file.c -shared -o file.so`
3. 在Python中导入和使用 `.so`
```python
from ctypes import *
clib = cdll.LoadLibrary("./file.so")
clib.add()
```
